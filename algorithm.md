## 动态规划
### 确认是动态规划问题
- 动态规划问题的一般形式就是求最值, 比如说让你求最长递增子序列呀，最小编辑距离
- 存在重叠子问题
- 具备最优子结构

### 解法
- 确定base case
- 明确状态
- 明确选择
- 找出状态转移方程
- 定义dp数组/函数

```python
# 初始化 base case
dp[0][0][...] = base
# 进行状态转移
for 状态1 in 状态1的所有取值：
    for 状态2 in 状态2的所有取值：
        for ...
            dp[状态1][状态2][...] = 求最值(选择1，选择2...)
```

- 不同的状态转移方程有不同的解法
- 写出动态规划之后或许还可以降维
  * 如果计算状态 `dp[i][j]` 需要的都是 `dp[i][j]` 相邻的状态，那么就可以使用空间压缩技巧
